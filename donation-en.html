Donate

This project is completely run by volunteers without commercial aims. Providing this website generates running costs, which are funded partly by donations, partly by the maintainers of the project. We want to remain independent from commercial interests and provide this service without advertisement. Therefore we need your financial support.

Bank transfer

FOSSGIS e.V. is a German non-profit organization supporting free and open source GIS software and open geodata. FOSSGIS manages bank transfer donations for this project. The bank transfer details can be found on the website of FOSSGIS e.V., you have to mention "OpenRailwayMap" as the reason for payment so that it can be assigned to this project.
